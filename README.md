# Client-Lead-System
A web-based client and lead management system built with Next.js that helps businesses track company details, contacts, sales opportunities, deal stages (open, won, lost), activities, and follow-ups. It provides a clean dashboard to monitor leads, manage customer interactions, and track sales progress efficiently.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Development

- **Framework**: Next.js 15+ (App Router)
- **Styling**: SCSS Modules
- **Database**: Hybrid MongoDB Driver (with Local JSON fallback)
- **Icons**: Lucide React
